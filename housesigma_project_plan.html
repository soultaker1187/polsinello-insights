# HouseSigma Duplication Project: Screen Breakdown & Features To duplicate the core functionality and user experience of HouseSigma, we need to focus on property search, detailed data visualization, and user engagement. Below is a breakdown of the essential screens. ## 1. Landing Page (Home) The entry point focused on immediate search and highlighting market trends. * **Key Features:** * Prominent search bar with filters (City, Address, MLS #). * Region selection (Ontario, British Columbia, Alberta). * Sections for "Just Sold", "Newly Listed", and "Hot Listings". * Quick links to market statistics and valuations. ## 2. Map-Based Search & Discovery The primary way users explore properties geographically. * **Key Features:** * Interactive map with property price pins. * Advanced filtering (Price, Bed/Bath, House Type, Sold/Active). * Split view: Map on one side, property cards on the other. * "Search in this area" functionality as the user pans. ## 3. Property Detail View The most data-heavy screen, providing a deep dive into a specific listing. * **Key Features:** * High-resolution photo gallery. * **HouseSigma Estimate:** AI-driven valuation. * **Sold History:** Crucial data showing what the house (and neighbors) sold for previously. * Market stats: Median price in area, days on market. * School rankings and neighborhood demographics. ## 4. Market Analytics & Statistics Differentiates the platform by providing professional-grade data to consumers. * **Key Features:** * Charts showing price trends over time. * Inventory levels and "Months of Inventory" metrics. * Comparison tools for different neighborhoods or cities. ## 5. User Profile & Favorites Personalization to keep users returning. * **Key Features:** * Saved searches and email alerts for new listings. * "Watchlist" for tracking specific properties. * Portfolio tracking for homeowners. --- **Next Steps:** I can start by designing a modern, clean version of the **Landing Page** based on the current site's layout but with some UX refinements.
